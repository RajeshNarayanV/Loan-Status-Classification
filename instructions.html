Read the instructions to run the app:

  1. Open the folder in VS Code.
  2. Open the new terminal.
  3. Create a Conda environment using the following command: conda create --name myenv.
  4. Activate it: conda activate myenv.
  5. Install the required libraries: pip install -r requirements.txt.
  6. Run the train_model.py file: python train_model.py.
  7. Run the app.py: streamlit run app.py.
  8. Check out the browser to view the web app.
